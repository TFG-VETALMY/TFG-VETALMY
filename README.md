# **VETALMY - Sistema de gestión veterinaria**
## Aplicación web diseñada para la gestión de una clínica veterinaria, en la cual se podrá acceder a la información de las mascotas, clientes y veterinarios.

<br>

## **Funcionalidades Principales**
### **- Gestión de usuarios:** Control de acceso para administradores.
### **- Gestión de clientes y mascotas:** Registro de propietarios y sus mascotas.
### **- Historial médico:** Registro de **enfermedades** y **vacunas** del animal.
### **- Gestión de citas médicas:** Registro de citas médicas para la mascota.
### **- Chat con el veterinario:** Apartado de mensajería con el veterinario.
<br>

## **Tecnologías usadas**
### **Web:** Angular
### **Móvil:** Angular + Capacitor
### **Base de datos:** MySQL + DBeaver + PostgreSQL + NestJS

<br>
